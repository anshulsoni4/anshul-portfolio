## About The Project

![image](https://user-images.githubusercontent.com/74638335/189617225-5ed0b448-8b32-48cb-b36d-63f19d2cb33d.png)

This is my Personal Portfolio made using react js, next js, styled-components (react js library), and design part from figma. This webpage contains all of my details you need to know, from project, achievements, experiences and many more.

## Getting Started

You can use this as a template and make your own personal porfolio (just give me some credits :D)

### Prerequisites

Install react, styled-components and other packages using yarn or npm. You need to have a basic idea of color palletes, responsive design and screen breakpoints. I made the UI design using the Figma tool, and other development with the react js.
* npm
  ```sh
  npm install npm@latest -g
  ```
